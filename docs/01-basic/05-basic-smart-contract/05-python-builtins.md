# Usage

-   Compile Contract

``` bash
algokit compile py contract.py
```

-   `Contract.approvoal.teal` & `Contract.clear.teal`

``` bash
alogkit compile py contract.py --output-arc32 --no-output-teal
```

-   `Contract.arc32.json`

``` bash
algokit generate client Contract.are32.json --output client.py
```
